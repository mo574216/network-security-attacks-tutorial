# Network Security Attacks Tutorial

This repository contains a GitHub Pages-ready tutorial on network security attacks for undergraduate students.

## Main files

- `index.md` — homepage
- topic pages in Markdown
- `glossary.md` — key terms
- `_config.yml` — basic Jekyll configuration

## Recommended workflow

1. Upload all files to a GitHub repository.
2. Enable GitHub Pages in repository settings.
3. Use the root branch or `/docs` directory as your Pages source.
4. Adjust theme, navigation, and wording as needed.

## Topics covered

- Foundations
- TCP/IP, spoofing, and DoS
- DNS cache poisoning
- Buffer overflow
- Malware
- Reconnaissance and scanning
- Password attacks
- Web attacks
- P2P security
- Botnets and DDoS
- Social engineering
- Mobile device security
- Defensive principles
- Labs
- Review questions
